Technical Jargon:

  1. GUI - Graphical User Interface, you click/drag graphical icons around.

  2. CLI - Command Line Interface, you give instructions, you give instructions to the computer by typing commands.

  3. Terminal - Also called "terminal emulator", is a program that gives you access to CLI on GUI.

  4. Prompt - The line of text that you see when you open a terminal it generally contains your username and machine's name(hostname) and most likely a $ sign.

  5. Directory - Same as a folder, we will call it a directory from now on.

What is the shell?

A shell is an interface for you, the user to interact with the operating system. It is called a shell as it is the outermost layer around the operating system. But you don't need to know much before you can start using the shell, we'll pick up more details about it as we move on!

Types of shell:

There are many shells available for Linux that can be installed, Zsh, Mksh, Pwsh, Csh, Fish, etc. But Bash is the one we'll be learning about in this series, as it is present by default on most Linux distributions(breeds of Linux :p) like Ubuntu, Fedora, RHEL, RockyLinux, OpenSUSE, etc. If you already have access to a Linux machine you can type echo $0 to find out which shell you're using
